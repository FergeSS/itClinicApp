# Экран авторизации

## Когда показывается
Экран отображается при первом запуске приложения на устройстве.

## Назначение
Предложить пользователю выбрать один из вариантов:
- войти через **GitHub**;
- продолжить работу **без авторизации**.

## Структура экрана
1. Вверху — название приложения.
2. По центру — кнопка **«Войти через GitHub»**.
3. Ниже — кнопка **«Продолжить без авторизации»**.
