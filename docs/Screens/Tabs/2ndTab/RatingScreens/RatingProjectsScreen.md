# Экран «Рейтинг» — вкладка «Проекты»

## Когда показывается
Открывается на вкладке **«Рейтинг»** в таббаре. По умолчанию отображается рейтинг **проектов**.

## Назначение
Показать список проектов, отсортированный по рейтингу, с возможностью поиска и фильтрации.

## Структура экрана

### 1) Заголовок и переключатель
- Заголовок: **«Рейтинг»**
- Переключатель разделов под заголовком:
  - **Проекты** (текущий)
  - **Студенты**

### 2) Поиск и фильтры
- Поле поиска (плейсхолдер: «Пример поиска»).
- Рядом иконки:
  - **фильтры**
  - **сортировка/изменение порядка** (стрелки).

### 3) Включённые фильтры
Под поиском отображаются активные теги-фильтры:
- **Commits**
- **Issues**
- **Pull Requests**

Эти теги показывают, какие фильтры сейчас применены к рейтингу.

### 4) Список рейтинга (скролл)
Каждый элемент списка содержит:
- место в рейтинге (номер слева)
- индикатор изменения позиции (стрелка вверх/вниз)
- карточку проекта: название, короткое описание, теги/стек (чипсы)
- значение рейтинга справа.

## Быстрое действие
Над таббаром отображается кнопка **«Личная статистика»**.

## Навигация
Внизу — таббар, активна вкладка **«Рейтинг»**.
