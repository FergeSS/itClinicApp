# Экран «Фильтры»

## Когда показывается
Открывается при настройке фильтров для рейтинга/списков (через кнопку фильтров). Позволяет быстро выбрать метрики и параметры, а также сохранить набор настроек как шаблон.

## Назначение
- Выбрать, **по каким метрикам** считается общая оценка.
- Настроить дополнительные параметры (веса, длительность, день недели и т.д.).
- Ограничить данные **по диапазону дат**.
- Применить фильтры или сохранить их как **шаблон**.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Фильтры»**.

### 2) Шаблоны
Горизонтальный список чипсов с вариантами шаблонов (например: **Нет**, **Коммиты**, **Пул Реквесты**, …).  
Выбор шаблона подставляет соответствующие настройки фильтров.

### 3) Метрики
Блок **«Метрики (отметь)»** с пояснением: пользователь выбирает метрики, из которых будет формироваться общая оценка.

Список метрик в виде чекбоксов (примерно):
- Commits
- Issues
- Pull Requests
- Оценка производительности
- Общее количество коммитов
- Завершённость задач
- Время жизни Pull Request
- Быстрые Pull Requests
- … (другие метрики)

У некоторых пунктов есть иконка **(i)** — подсказка/описание метрики.

### 4) Дополнительные параметры
Настройки, которые зависят от выбранных метрик, например:
- поле веса (пример: **«2,5 часа»**)
- дополнительные критерии (например, «Изменимость кода», «Владение кодом»)
- выбор дня недели (пример: **«Четверг»**)

### 5) Диапазон дат
Блок **«Диапазон дат»**:
- поле выбора периода (интервал дат) с иконкой календаря
- кнопка **«Очистить»** — сбрасывает выбранный период

### 6) Действия
Внизу экрана две кнопки:
- **Применить** — применяет выбранные фильтры.
- **Сохранить шаблон** — сохраняет текущие настройки как новый шаблон.
