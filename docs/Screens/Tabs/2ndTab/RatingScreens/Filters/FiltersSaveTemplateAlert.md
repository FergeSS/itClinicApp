# Экран «Фильтры» — модальное окно «Сохранить шаблон»

## Когда показывается
Открывается поверх экрана **«Фильтры»** при нажатии на кнопку **«Сохранить шаблон»**. Фон затемняется, остальные элементы экрана становятся неактивными.

## Назначение
Сохранить текущий набор выбранных фильтров как шаблон, чтобы быстро применять его в будущем.

## Содержимое модального окна
- Заголовок: **«Сохранить шаблон»**
- Кнопка закрытия (**×**) в правом верхнем углу.
- Поле ввода названия (плейсхолдер): **«Введите название шаблона»**
- Кнопка действия: **«Сохранить»**

## Поведение
- По нажатию **«Сохранить»** шаблон создаётся и становится доступен в списке шаблонов.
- По нажатию **×** окно закрывается без сохранения.
