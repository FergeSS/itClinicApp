# Экран «Настройки» — настройка экрана статистики

## Когда показывается
Открывается из экрана статистики по действию **«Настройки экрана»**.

## Назначение
Позволяет настроить, какие блоки (вкладки) отображаются на экране статистики и в каком порядке.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Настройки»**.
- Подзаголовок/пояснение: пользователь выбирает вкладки для личной и проектной статистики и задаёт их порядок.

### 2) Список используемых вкладок
Карточки вкладок, которые сейчас включены (например):
- **Коммиты**
- **Issue**
- **Pull Request**
- **Быстрые Pull Request**
- **Изменчивость кода**
- **Владение кодом**

Для каждой вкладки доступны действия:
- **перетаскивание** (иконка “ручки”) — изменить порядок.
- **удалить/скрыть** (иконка корзины) — убрать вкладку из отображаемых.

### 3) Блок «Неиспользуемые»
Список вкладок, которые скрыты и не показываются на экране статистики (например: **Доминирующий день недели**).

Действие:
- кнопка **«+»** — добавить вкладку обратно в используемые.
