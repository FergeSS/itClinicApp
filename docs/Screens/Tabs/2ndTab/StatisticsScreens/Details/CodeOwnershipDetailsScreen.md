# Экран «Владение кодом» — подробно

## Когда показывается
Открывается из экрана статистики по кнопке **«Подробнее»** в блоке «Владение кодом».

## Назначение
Показать распределение владения кодом между участниками за выбранный период и дать сводные значения/рейтинг.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Владение»**.

### 2) Параметры отчёта
- **Выбор репозитория** (выпадающий список).
- **Выбор периода** (интервал дат “с…по…”).
- **Выбор участника** (выпадающий список) — отображается, когда экран открыт в контексте проекта/команды.

### 3) Основная визуализация
- Кольцевая диаграмма распределения владения кодом (доли участников в процентах).
- Легенда со списком участников и объёмом (у текущего пользователя пометка **«(Вы)»**).

### 4) Сводные показатели
- **Всего строк** (общее количество строк кода).
- **Таблица участников**:
  - участник
  - количество изменений
  - количество строк
  - дополнительные показатели/дельта (если отображается).

### 5) Рейтинг и оценка
- **Место в рейтинге**.
- Итоговая **оценка владения кодом**.
