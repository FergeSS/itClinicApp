# Экран «Быстрые PR» — подробно

## Когда показывается
Открывается из экрана статистики по кнопке **«Подробнее»** в блоке «Быстрые Pull Requests».

## Назначение
Показать динамику и сводные показатели по быстрым PR, сравнение участников, а также список конкретных PR, которые попали в категорию “быстрых”.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Быстрые PR»**.

### 2) Параметры отчёта
- **Выбор репозитория** (выпадающий список).
- **Выбор периода** (интервал дат “с…по…”).
- **Выбор участника** (выпадающий список) — отображается в контексте проекта/команды.
- **Выбор периода для быстроты** (например, «Период: 1 день / 1н»).

### 3) График
- Блок **«График быстрых PR»** с динамикой по датам.

### 4) Сводные показатели
- **Всего быстрых PR**.
- **Место в рейтинге**.
- Список **«Количество быстрых Pull Request»** по участникам (с пометкой **«(Вы)»** у текущего пользователя).

### 5) Список быстрых Pull Requests
- Заголовок блока: **«Список быстрых Pull Requests»**
- Действие **«Смотреть все»** — открывает полный список.
- Карточки PR с основными данными:
  - название PR
  - статус (например, **closed**)
  - автор/создатель
  - назначенные участники (если есть)
  - дата создания и дата закрытия
  - мини-статистика (иконки комментариев/коммитов и т.п.)
  - ссылка на PR

### 6) Итоги
- Доля быстрых PR от общего числа (например, **«58% быстрых PR из всех»**).
- Участник-лидер (кто сделал больше всего быстрых PR).
- Итоговая **оценка быстрых Pull Request**.
