# Экран «Коммиты» — подробно

## Когда показывается
Открывается из экрана статистики по кнопке **«Подробнее»** в блоке «Коммиты».

## Назначение
Показать подробную статистику по коммитам за выбранный период: динамику, средние значения, изменения строк, наиболее изменяемые файлы и список коммитов.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Коммиты»**.

### 2) Параметры отчёта
- **Выбор репозитория** (выпадающий список).
- **Выбор периода** (интервал дат “с…по…”).
- **Выбор участника** (выпадающий список) — отображается в контексте проекта/команды.

### 3) Сводные показатели
Карточки с ключевыми значениями:
- **Всего коммитов**
- **Место в рейтинге**

### 4) График коммитов
- Блок **«График коммитов»** с динамикой по датам.

### 5) Дополнительные метрики
- **Среднее количество коммитов в день**
- **Добавлено строк**
- **Удалено строк**
- График **изменения строк** по датам (добавлено/удалено).
- **Макс. коммитов/день**
- **Мин. коммитов/день**

### 6) Топ изменяемых файлов
- Блок **«Топ-10 изменяемых файлов»**:
  - путь к файлу
  - количество изменений
  - статистика по строкам (например, `+… / -…`)
  - тип изменения (например, *added / modified / removed*).

### 7) Список коммитов
- Блок **«Список коммитов»**:
  - автор
  - сообщение коммита
  - дата/время
  - ссылка на коммит
  - краткая статистика по изменениям (например, `+… / -…`).

### 8) Итоги
- **Место в рейтинге**
- **Место в команде**
- Итоговая **оценка коммитов**
