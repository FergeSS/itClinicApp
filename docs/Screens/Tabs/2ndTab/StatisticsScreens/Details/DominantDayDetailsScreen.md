# Экран «ДДН» — доминирующий день недели (подробно)

## Когда показывается
Открывается из экрана статистики по кнопке **«Подробнее»** в блоке «Доминирующий день недели».

## Назначение
Показать распределение активности по дням недели за выбранный период и выделить самый активный и самый неактивный дни.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«ДДН»**.

### 2) Параметры отчёта
- **Выбор репозитория** (выпадающий список).
- **Выбор периода** (интервал дат “с…по…”).
- **Выбор участника** (выпадающий список) — отображается, когда экран открыт в контексте проекта/команды.

### 3) Основная визуализация
- Кольцевая диаграмма распределения активности по дням недели.
- Легенда со списком дней недели и количеством действий по каждому дню.

### 4) Итоги
- Блок **самый активный день недели** (например, **«ПОНЕДЕЛЬНИК»**).
- Блок **самый неактивный день недели** (например, **«ЧЕТВЕРГ»**).
- Итоговая **оценка доминирующего дня недели**.
