# Экран «Политика конфиденциальности»

## Когда показывается
Открывается из экрана **«Информация»** по пункту **«Политика конфиденциальности»**.

## Назначение
Показать текст политики конфиденциальности приложения.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Политика конфиденциальности»**.

### 2) Контент
- Текстовый блок с содержимым политики (прокручивается при необходимости).
