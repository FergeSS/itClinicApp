# Экран «Настройки»

## Когда показывается
Открывается из экрана **«Информация»** по пункту **«Настройки»**.

## Назначение
Позволяет изменить основные параметры приложения: язык интерфейса и тему оформления.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Настройки»**.

### 2) Параметры
- **Язык** — выбор языка интерфейса (например, значение: **RU**).
- **Тема** — выбор темы оформления (например, значение: **Светлая**).
