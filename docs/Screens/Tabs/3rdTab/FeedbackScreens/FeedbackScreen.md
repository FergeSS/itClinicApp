# Экран «Обратная связь»

## Когда показывается
Открывается из экрана **«Информация»** по пункту **«Обратная связь»**.

## Назначение
Собрать предложения и замечания пользователя, чтобы улучшать приложение.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Обратная связь»**.

### 2) Текст-пояснение
- Заголовок блока: **«Помогите нам стать лучше»**
- Описание:
  «Напишите, если что-то работает не так, чего не хватает или что было бы удобно добавить. Мы читаем каждое сообщение.»

### 3) Поле ввода
- Большое текстовое поле для сообщения.
- Плейсхолдер с примером (Например: «не открывается экран проекта», «хочу видеть дедлайны на главной»…).

### 4) Действие
- Кнопка **«Отправить»** — отправляет сообщение обратной связи.
