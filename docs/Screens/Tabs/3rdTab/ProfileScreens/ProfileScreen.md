# Экран «Профиль»

## Когда показывается
Открывается из экрана **«Информация»** при нажатии на карточку пользователя.

## Назначение
Показать данные профиля пользователя, информацию о команде и проекте, а также дать возможность выйти из аккаунта.

## Структура экрана

### 1) Шапка
- Кнопка **назад** (стрелка).
- Заголовок: **«Профиль»**.

### 2) Карточка пользователя
Отображает:
- аватар
- имя и фамилию
- email
- телефон
- иконку **редактирования** (карандаш) для изменения данных профиля.

### 3) Команда
Блок **«Команда»** со списком участников (имя + роль).  
В заголовке блока есть иконка **редактирования** (карандаш) для редактирования своей роли.

### 4) Проект
Блок **«Проект»** с названием/описанием текущего проекта пользователя.

### 5) Выход из аккаунта
- Пункт **«Logout»** — завершает текущую сессию пользователя.
