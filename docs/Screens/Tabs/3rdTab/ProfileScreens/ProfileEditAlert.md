# Экран «Профиль» — модальное окно «Редактировать»

## Когда показывается
Открывается поверх экрана **«Профиль»** при нажатии на иконку редактирования (карандаш) в карточке пользователя. Фон затемняется, основной экран становится неактивным.

## Назначение
Позволяет изменить данные профиля и настройки уведомлений.

## Содержимое модального окна
- Заголовок: **«Редактировать»**
- Кнопка закрытия (**×**) в правом верхнем углу.
- Поля ввода:
  - **Имя**
  - **Фамилия**
  - **Отчество**
  - **Почта**
  - **Номер телефона**
- Чекбокс:
  - **«Получать оповещения об изменении статуса проектов, в которых Вы участвуете»**
- Кнопка действия:
  - **«Сохранить»**

## Поведение
- **×** — закрывает окно без сохранения.
- **Сохранить** — применяет изменения и закрывает окно.
