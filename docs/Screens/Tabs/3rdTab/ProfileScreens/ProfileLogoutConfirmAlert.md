# Экран «Профиль» — модальное окно подтверждения выхода

## Когда показывается
Открывается поверх экрана **«Профиль»** после нажатия на пункт **Logout**. Фон затемняется.

## Назначение
Подтвердить действие выхода из аккаунта и избежать случайного разлогина.

## Содержимое модального окна
- Заголовок/вопрос: **«Выйти?»**
- Кнопка закрытия (**×**) в правом верхнем углу.
- Две кнопки:
  - **«Нет»** — отменяет выход и закрывает окно.
  - **«Да»** — подтверждает выход из аккаунта.

## Поведение
- **Нет** / **×** → окно закрывается, пользователь остаётся на экране профиля.
- **Да** → пользователь выходит из аккаунта.
