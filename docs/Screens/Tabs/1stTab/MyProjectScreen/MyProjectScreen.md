# Экран «Мой проект»

## Когда показывается
Открывается, когда пользователь переходит в свой проект (например, через пункт **«Мой проект»**).

## Назначение
Показать подробную информацию о личном проекте: основные данные, статус, описание, требования и состав команды, а также дать быстрые действия (например, открыть GitHub и статистику проекта).

## Структура экрана

### 1) Заголовок
- Заголовок экрана: **«Мой проект»**.

### 2) Карточка проекта
Содержит ключевые данные:
- Дата (слева).
- Название проекта.
- Иконка обновления/перезагрузки данных.
- Плашки/поля с датами:
  - **Срок записи на проект**
  - **Срок реализации проекта**
- Информация о заказчике.

### 3) Дополнительная информация
- Контакты/заказчик (в виде небольших плашек).
- Теги/стек проекта (в виде чипсов).
- Статус: **«Назначена команда»**.

### 4) Описание проекта
- Заголовок: **«Описание проекта»**
- Текстовое описание проекта.

### 5) Требования проекта
- Заголовок: **«Требования проекта»**
- Нумерованный список требований.

### 6) Требования для исполнителей
- Заголовок: **«Требования для исполнителей»**
- Чипсы с навыками/требованиями.

### 7) Команда
- Карточка **«Команда»** со списком участников и их ролями.

### 8) Быстрые действия
Справа внизу — группа кнопок:
- **Все проекты**
- **GitHub**
- **Статистика проекта**
