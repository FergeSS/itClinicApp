# Экран «Проекты» — модальное окно «Заказчикам»

## Когда показывается
Поверх экрана «Проекты» открывается модальное окно для заказчиков (фон затемняется, основное содержимое становится неактивным).

## Назначение
Дать заказчику возможность оставить заявку на сотрудничество/создание проекта.

## Содержимое модального окна
- Заголовок: **«Заказчикам»**
- Кнопка закрытия (**×**) в правом верхнем углу.
- Описание:
  «Если у Вас есть запрос на сотрудничество и создание проекта, заполните онлайн-заявку. Наш представитель свяжется с вами в ближайшее время».
- Поля ввода:
  - **Имя**
  - **Email** (пример в поле: `st000000@student.spbu.ru`)
- Основная кнопка действия: **«Отправить»**

## Техническая реализация

### Компонент

**Файл:** `composeApp/src/commonMain/kotlin/com/spbu/projecttrack/projects/presentation/components/SuggestProjectAlert.kt`

**Composable функция:**
```kotlin
@Composable
fun SuggestProjectAlert(
    isVisible: Boolean,
    onDismiss: () -> Unit,
    onSubmit: (String, String) -> Unit, // name, email
    modifier: Modifier = Modifier
)
```

### API запрос

#### Отправка заявки заказчика

**Эндпоинт:** `POST /email/send-request`

**Запрос:**
```json
{
  "name": "string",
  "email": "string"
}
```

**Ответ:**
- Успешный ответ (200) — заявка отправлена
- Ошибка (400/500) — ошибка отправки

**Реализация:**
- Эндпоинт: `ApiConfig.Public.EMAIL_SEND_REQUEST = "/email/send-request"`
- Публичный эндпоинт (не требует авторизации)

### Состояние формы

**Локальное состояние:**
```kotlin
var name by remember { mutableStateOf("") }
var email by remember { mutableStateOf("") }
```

**Валидация:**
- В текущей версии валидация не реализована
- Рекомендуется добавить проверку формата email перед отправкой

### Обработка отправки

**Поток:**
1. Пользователь заполняет поля "Имя" и "Email"
2. Нажимает кнопку "Отправить"
3. Вызывается `onSubmit(name, email)`
4. Алерт закрывается (`onDismiss()`)
5. Данные отправляются на сервер через API

**Реализация отправки:**
- Обработчик `onSubmit` передаётся из родительского компонента
- В `MainScreen.kt` или `ProjectsScreen.kt` реализуется отправка HTTP запроса
