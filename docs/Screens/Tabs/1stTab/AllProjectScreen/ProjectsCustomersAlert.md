# Экран «Проекты» — модальное окно «Заказчикам»

## Когда показывается
Поверх экрана «Проекты» открывается модальное окно для заказчиков (фон затемняется, основное содержимое становится неактивным).

## Назначение
Дать заказчику возможность оставить заявку на сотрудничество/создание проекта.

## Содержимое модального окна
- Заголовок: **«Заказчикам»**
- Кнопка закрытия (**×**) в правом верхнем углу.
- Описание:
  «Если у Вас есть запрос на сотрудничество и создание проекта, заполните онлайн-заявку. Наш представитель свяжется с вами в ближайшее время».
- Поля ввода:
  - **Имя**
  - **Email** (пример в поле: `st000000@student.spbu.ru`)
- Основная кнопка действия: **«Отправить»**
